#include <ViennaRNA/subopt.h>

Returns list of subopt structures or writes to fp.

This function produces all suboptimal secondary structures within 'delta' * 0.01 kcal/mol of the optimum, see [wuchty:1999.] The results are either directly written to a 'fp' (if 'fp' is not NULL), or (fp==NULL) returned in a #vrna_subopt_solution_t * list terminated by an entry were the 'structure' member is NULL.

See also
vrna_subopt_zuker()
